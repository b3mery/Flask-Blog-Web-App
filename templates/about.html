About Me

This is what I do.

This is a fake blog site. It was developed for 100 days of code on Udemey. The site uses a bootstrap blog template as a scaffold. The site is run using Python Flask and is connected to a relational database.

The site has real production ready functionality, below is a list of the features:

  • Login and Registration functionality
  • Password Hashing
  • Dynamic nav based on current authentication.
  • Users can create, edit or delete their posts.
  • Admin accounts can create, edit or delete any post.
  • Login users can post comments on posts.
