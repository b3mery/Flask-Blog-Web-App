About

Some information about the site.

This is a fake blog site. It was developed for 100 days of code on Udemey. The site uses a Start Bootstrap Clean Blog template as a scaffold, checkout Start Bootstrap to see their other awesome templates.

The site is run using Python Flask and is connected to a relational database. It has real production ready functionality, below is an overview of the functionality:

  • User Login and Registration:
    • Minimun 8 charater Password
    • Password Hashing
    • Unique email
    • Form error feeback, such as incorrect password, email does not exist, email is already used, etc.
  • Dynamic Navigation based on current user authentication.
  • Dynamic options for users to:
    • Create a new blog.
    • Edit an existing blog.
    • Delete an existing. With delete confrimaiton popup modal.
    • Comment on their or another users post.
  • Admin accounts can create, edit or delete any users blog post. Note: Admin account will need to be created/updated in the database.
